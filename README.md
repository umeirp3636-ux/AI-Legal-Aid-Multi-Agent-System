# AI-Legal-Aid-Multi-Agent-System

## Overview

LexiGuard AI is an AI-powered Multi-Agent Legal Assistant built using IBM watsonx technologies.

## Features

- Legal Information Retrieval (RAG)
- Contract Review
- Compliance Checking
- Case Research
- Conversational AI
- Document Analysis

## Technology

- IBM watsonx Orchestrate
- IBM Langflow
- IBM Granite Models

## Architecture

Multi-Agent workflow with RAG and Document Processing.
